## **구성 내용 💭**

- `앨범, 이벤트, 로그인 박스`로 나누어 크게는 총 3개의 박스로 구성하였습니다.
- `display: flex 대신 grid를 사용`하였고, 맞지 않는 라인은 적절히 `여백과 이미지 크기를 조절`하여 맞추었습니다.
- 최신 앨범의 가수 이름은 투명한 배경을 맡는 박스와 텍스트를 맡는 박스로 구성하고, `position: absolute` 를 사용였습니다.
